# How to change site settings

You can change site name theme color description and player options

Open this file to change settings
src/components/shiopa/config.shiopa.ts

## Settings list

- text
This is your site logo name

- defaultHue
This is the theme color number between 0 and 360

- description
This is the text for search engines

- defaultServer
This is the starting video server
Options are vidzeeWorks vixsrc vidlink and icefy

- useVidstack
Set to true for vidstack player
Set to false for custom HTML5 player

- showTrending
Set to true to show trending list on home page
Set to false to hide it
